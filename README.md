# YGPA PortSight Slides

여수광양항만공사 PortSight 제안 슬라이드를 Figma 노드 기반 HTML/CSS 레이어로 웹 퍼블리싱한 정적 Vite 프로젝트입니다.

텍스트와 레이아웃 박스는 DOM/CSS로 렌더링하고, 대시보드 목업·아이콘·벡터처럼 복합 그래픽으로 관리되는 요소만 개별 에셋으로 사용합니다.

## Scripts

```bash
npm install
npm run dev
npm run build
```

The page is optimized as a 16:9 slide deck with responsive scaling.
